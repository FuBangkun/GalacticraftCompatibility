This mod resolves compatibility issues between Galacticraft addons by modifying configuration files.
After the game loads, you will enter the configuration interface. Make your selections based on your actual situation, and then click the button to exit and restart the game to complete the configuration.
The configuration interface will only appear once. To display it again, you must set "Display GCC Configuration Interface" to true in the configuration file or enter the "/gcc" command in the world.
However, the registration of The Sol fluids is a matter of luck.
In addition, this mod also provides:
- Modifications to shaders, the new main menu for Galaxy Space, and advanced rocket craft.
- Automatic disabling of the beta version warning for Interstellar: Exoplanets.
